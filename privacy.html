🔒 Privacy Policy

Last updated: 2025

Overview

Trim Finder is a static website with no backend server, no user accounts, and no data collection. This privacy policy explains what minimal data handling takes place when you use the site.

Data We Do Not Collect

  • We do not collect your name, email, or any personal information
  • We do not use cookies for tracking
  • We do not run analytics or user tracking scripts
  • We do not sell or share any data with third parties

Local Storage

Trim Finder uses your browser's localStorage to save your checklist progress (which trims you've marked as found). This data is stored entirely on your own device and is never transmitted to any server. You can clear it at any time by clicking "Reset All" on the checklist or by clearing your browser's site data.

External Resources

The site loads fonts from Google Fonts (fonts.googleapis.com). This is a standard web practice — Google may log the font request as part of their service. See Google's Privacy Policy for details.

Trim textures are loaded from GitHub (raw.githubusercontent.com). GitHub may log standard server access requests. See GitHub's Privacy Statement for details.

Hosting

This site is hosted on GitHub Pages. GitHub may collect standard server logs including IP addresses as part of their hosting service. This is outside the control of Trim Finder. See GitHub's privacy statement for more information.

Changes to This Policy

If this privacy policy is ever updated, the "last updated" date at the top of this page will change. Continued use of the site after changes constitutes acceptance of the updated policy.

Contact

If you have any questions about this privacy policy, please use the contact page.
