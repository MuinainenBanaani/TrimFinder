✉ Contact

Get in touch with the creator of Trim Finder

GitHub (Preferred)

The best way to report bugs, suggest features, or ask questions is through the GitHub repository:

github.com/MuinainenBanaani/TrimFinder/issues

Open a new Issue and describe your question or problem. This keeps everything organized and visible to others who might have the same question.

Minecraft / NameMC

You can also find the creator on Minecraft and NameMC:

fi.namemc.com/profile/MuinainenBanaani.1

What to Include

If you're reporting a bug, please include:

  • Which browser and operating system you're using
  • What you expected to happen
  • What actually happened
  • A screenshot if possible

Response Time

This is a personal project maintained in spare time. Responses may take a few days. Thank you for your patience!
